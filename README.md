# Spark Glow Animation Website

This project is a web application that uses React and TypeScript to create a spark glow animation effect.

## Features

- Spark glow animation effect
- Responsive design
- Modern UI

## Installation

1. Clone the repository
2. Install dependencies
```bash
npm install
```

## Usage

```bash
npm run dev
```
